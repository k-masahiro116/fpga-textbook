# FPGA教科書

FPGAを基礎から学ぶための教科書プロジェクトです。

## 概要

このリポジトリは、FPGAの基礎理論から実践的な設計まで、体系的に学べる教科書を提供します。

## 構成

- `index.html`: トップページ（章一覧）
- `chapters/`: 各章の本文
- `assets/`: 共有スタイルなどの共通リソース

## 目次

### 第1章: FPGA基礎理論
- FPGAとは
- FPGAの歴史と発展
- ASICとの比較
- FPGAの内部アーキテクチャ
- デジタル論理の基礎
- FPGA設計フロー

### 第2章: HDL入門
- HDLとは
- 記述レベルの概要
- 開発ツールの流れ
- VerilogとVHDLの違い
- 最初の記述例
- 次に学ぶこと

### 第3章: 実践的な設計手法
- 目的と全体像
- 設計方針の決め方
- モジュール分割と再利用
- テストベンチ設計
- デバッグと改善の流れ

### 第4章: タイミング設計
- タイミング設計の重要性
- クリティカルパスと制約
- クロック設計の基本
- セットアップ/ホールドの考え方
- タイミング解析の進め方

## 閲覧方法

ブラウザで `index.html` を開くと、章一覧から各章に移動できます。
